# PokéDex App

Dependecias: Ir al folder del proyecto y escribir el siguiente comando:

  * npm install
 
 y es todo.
 
 ## Cómo funciona?
  * Presiona sobre el Pokémon que más te agrade y mira los detalles acerca de él.
  * Conoce nuevos Pokémon con el botón de "Shuffle"
